项目简介¶

LaunchyQt是什么¶

Launchy是一款小工具，旨在帮助您忘记开始菜单，桌面图标和文件管理器。只需几次按键即可搜索并启动应用程序，文档，项目文件和文件夹！

我一直在使用Launchy多年，并发现这个项目自2010年以来一直没有更新。我喜欢这个工具，并希望维护这个项目。

核心功能¶

  • 快速搜索并启动程序

  • 计算器（由Calcy插件提供）

  • 开放的插件系统

  • 用C++开发插件

  • 用Python中开发插件
